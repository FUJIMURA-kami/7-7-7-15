7月7日

  1. ウィンナーと野菜のポトフ風煮込み

    【材料】

    • ウィンナー4本
    • たまねぎ 1個(くし切り)
    • にんじん 1本(乱切り)
    • じゃがいも 3~4個(1口大に切る)
    • 水800~1000ml
    • コンソメキューブ 2個(または顆粒スープの素 大さじ1)
    • 塩・こしょう 適量
    • オリーブオイル(またはサラダ油) 大さじ1

    【作り方】

    1. 鍋にオリーブオイルを熱し、ウィンナーを軽く炒めて香ばしさを出す。
    2. たまねぎ、にんじん、じゃがいもを加えてさっと炒める。
    3. 水を加え、コンソメキューブを入れて中火にかける。
    4. 沸騰したらアクを取り、蓋をして弱火で約20~30分、野菜が柔らかくなるまで煮込む。
    5. 塩・こしょうで味を調えたら完成。

  2. サラダ

    【材料】

    • 生たまご 3個
    • レタス 適量
    • きゅうり 適量
    • トマト 適量

    【作り方】

    1. 鍋に水を入れ、卵がかぶるくらいの量にする。
    2. 沸騰したら卵をそっと入れ、8~10分茹でる。
    3. 茹で上がったら冷水にとり、殻をむく。
    4. ゆで卵1つを薄くカットし、レタスやきゅうり、トマトで簡単に。
    5. 余ったゆで卵は冷蔵保存し、明日以降のサラダで活用。

  3. みそ汁 (作り置きします)

    【材料】

    • たまねぎ 1/2個(薄切り)
    • じゃがいも 1~2個(小さめ1口大)
    • にんじん 1/2本(薄切り)
    • 乾燥わかめ 適量
    • 豆腐 1丁(さいの目切り)
    • ねぎ 1本(小口切り)
    • みそ 適量
    • だし汁(水＋顆粒だしなどで) 1.5~2L

    【作り方】

    1. 鍋にだし汁を入れ、たまねぎ、じゃがいも、にんじんを入れて火にかける。
    2. 野菜が柔らかくなったら豆腐を加え、軽く温める。
    3. 火を弱めてみそを溶かし入れる。
    4. 最後にわかめとねぎを加え、ひと煮立ちさせたら火を止める。

  4. ご飯

    ・3人前を炊きあげ、翌日の昼食にも使います。

  1. 野菜たっぷり焼きそば

    【材料】

    • 焼きそば麺 1人前分
    • たまねぎ 1個(薄切り)
    • にんじん 1本(細切り)
    • じゃがいも 1~2個(細切りまたは千切り)
    • ウィンナー 2本(斜め薄切り)
    • サラダ油またはごま油 大さじ1~2
    • 焼きそばソース (市販のもの)またはお好みの味付け
    • 塩・こしょう 適量

    【作り方】

    1. フライパンに油を熱し、ウィンナーを炒める。
    2. たまねぎ、にんじん、じゃがいもを加えてしんなりするまで炒める。
    3. 焼きそば麺をほぐしながら加え、全体をよく混ぜ炒める。
    4. 焼きそばソースを加えて味を調え、塩・こしょうで調える。
    5. 全体に火が通ったら完成。

  2. サラダ

    ・レタス、きゅうり、トマトなどお好みの野菜で簡単に。


  3. みそ汁

    ・7/7にまとめて作ったみそ汁を温め食べる。


  4. ご飯

    ・3人前を炊きあげ、翌日の昼食にも使います。

    • 牛乳 500ml
    • チーズ（ピザ用やとろけるタイプ） 100〜150g
    • バターまたはサラダ油 大さじ1
    • 小麦粉 大さじ2
    • 塩・こしょう 適量
    • お好みでパン粉（焼き色用）

    【作り方】

    1. フライパンにバターまたは油を熱し、たまねぎ、にんじん、ウィンナーを炒める。

    5. 塩・こしょうで味を調える。
    6. 耐熱皿に移し、上からチーズをたっぷりのせる。
    7. お好みでパン粉を散らし、200℃のオーブンで20分ほど焼き色がつくまで焼く。

  1. 豚肉と野菜の生姜焼き風炒め

  1. 鮭のホイル焼き

    【材料】

    • 鮭切り身 3切れ
    • キャベツ 1/2玉（ざく切り）
    • ピーマン 3個（細切り）

    • レモン（お好みで）
    • 塩・こしょう 適量
    • オリーブオイルまたはバター 大さじ1
    • アルミホイル

    【作り方】

    1. アルミホイルを3枚用意し、それぞれにキャベツ、ピーマン、じゃがいもを均等にのせる。

    3. オリーブオイルまたはバターを少量ずつのせる。
    4. アルミホイルで包み、しっかり閉じる。
    5. オーブントースターまたはオーブンで180℃で約20分焼く。
    6. お好みでレモンを絞って召し上がれ。
